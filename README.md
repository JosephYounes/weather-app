# React Weather App

A simple front-end-only weather application built with React and React Router.  
Users can search for a city and view the current weather using the public Open-Meteo API.

## Features

- City search with a controlled input
- Asynchronous data fetching from an external API
- Simple routing with React Router
- Component-based UI with basic styling

## Tech Stack

- React
- React Router DOM
- Vite (build tooling)
- Open-Meteo API
